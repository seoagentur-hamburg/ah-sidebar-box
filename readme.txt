== Description ==

Creates a new and simple to use widget that adds a tabbed sidebar box with recent posts, last comments, categories, popular posts, a tag cloud and the archives to the sidebar. No Options. This widget is perfect for light-colored themes.

To use the widget, go to Appearance -> Widgets and add "AH Sidebar Box" widget.

== Installation ==

1. Upload contents of the directory to /wp-content/plugins/ (or use the automatic installer)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In Appearance->Widgets, add Evolution Sidebar Box widget to any sidebar.
